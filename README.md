# Slot Game
## Back-End (Node.js) Project

### Requirements:
- OOP
- TypeScript


### Installation
1. Clone the repository:
```js
git clone <repository-url>
```
2. Navigate into the project directory:
```js
cd slot-game
```
3. Install dependencies:
```js
npm install
```
4. Run the project
```js
cd slot-game/src
npm start
```
5. You can uncomment the code in main.ts and try freeSpins execution
6. **Play Responsibly!**
